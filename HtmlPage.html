Hotel Management System

As part of a 2nd year project containing four members we developed a hotel management system that had 2 kinds of users, Administrators and people who wanted to book a room in the We researched other similar systems and implemented the best features of those systems into our own. We drew storyboards of the system detailing what each screen should allow a user to do.

We decided that people trying to book a room didnt have to sign in right away as this would annoy users. This allowed users to browse available rooms before proceeding with a booking. Administrators of the system could remove rooms from the system and delete bookings for a user.

We used a JDBC database to store data about users and rooms while java swing was used to design the user interface. This project gave us a great insight into the development of a system and each of us benefited greatly from this as the project group received a grade of 94 which was the highest grade in the year.

Android Application used to discover local restaurants

As part of a 3rd year project consisting of two team members, we decided to create an android application that allowed users to firstly, find a restaurant that was closest to them, secondly, allow a user to book a table in that restaurant as if they were booking a seat in the cinema and thirdly, remove the hassle involved for restaurants in trying to make their own app for their customers.

The android application was designed to allow users to find restaurants that were closest to them based on their current location. However their search criteria wasn't limited to this as they could also search for restaurants based on the options we provided them with such as searching by name or searching for wheelchair accessible restaurants. Each restaurant presented to the user in the application had their own section purely about their restaurant which provided users with useful information.

Booking a table at a restaurant was the key functionality to this project and it proved to be a quite a challenge. Myself and my project partner wanted the booking of a table to be easy like booking a cinema ticket for a film you want to see. With that in mind we decided to allow a user to book a table by simply selecting the table they wanted from a floor plan that was designed by the restaurant themselves. Once a user had confirmed their booking the user would recieve an email and a reminder was automatically set on the android device.

A web application was developed along with the android application to allow restaurants to be in charge of what users of the android application saw. We kept track of restaurants location by asking them to show us on a google map where their restaurant was located and in the background we saved the longitude and latitude of the location so we could use it to determine how close restaurants were to users using the harversine algorithm (an algorithm used to determine the distance between to points). The most important functionality of the web application was allowing each restaurant create one or more floor plans as this was how users made bookings on restaurants.

Home Security Application

My fourth year project is going to be an android application that allows users to turn their old android devices into security cameras. Users will control their old android devices by sending commands to them from their day to day device. Commands will be sent using Azures notification hubs, which will communicate with androids PNS (Push Notification Service), Google Cloud Messaging. Commands such as enabling/disabling motion detection and turning on or off the flashlight on the device will be available to users.

Upon detected motion the user will recieve a picture showing the cause of the detected motion in an email. Users can then choose from a list of thier available security cameras to view the live feed coming from these devices. Live video feeds will be available from Azure Media Services which will be responsible for producting multiple bit rates for a video feed. This allows for different quality of videos to be streamed thus allowing videos to be available to more users.
