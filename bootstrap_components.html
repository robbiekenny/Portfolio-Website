Components Used

Robert Kenny - Computing

This website didn't build on a template. The following is a list of key bootstrap components used in the website.

  • Scroll Spy - This feature was used in the navigation bar to highlight a section in the navigation bar that the user is on

  • Navigation bar - Used to allow a user to navigate through the page

  • Accordion - Used in the projects section of the website to hide or show a particular project

  • Modal - A contact modal was used that appears to allow users send me a message

  • Tool tip - Tool tips were used on the social icons on the right hand side of the navigatin bar

  • Pop over - A pop over was used on the email icon on the right hand side of the navigation bar

Along with these elements I used CSS classes from bootstrap that made elements responsive. The website is hosted on Azure and can be accessed here.
